## 📈 Roadmap

### MVP (2 semanas)
- Loja ↔ Cliente: chat básico
- Loja ↔ Motoboy: chat básico
- Motoboy ↔ Cliente: chat básico
- Real-time com Socket.io ✅
- Status delivery/read ✅
- Persistência em MongoDB ✅

### Phase 2 (2-3 semanas)
- Attachments (fotos, location)
- Push notifications
- Online status
- Message search
- Auto-delete

### Phase 3 (3-4 semanas)
- End-to-end encryption
- Voice messages
- Video call
- AI suggestions
- Analytics
- Multi-language
- File sharing
