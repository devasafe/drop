## 🐛 PROBLEMA? SOLUÇÃO

### Não vejo dropdown
```
→ Verifique se tem múltiplos roles
→ Se só tem 1 role, dropdown não aparece (normal)
```

### Nada acontece ao clicar
```
→ Abre DevTools (F12)
→ Aba "Console"
→ Procura por erros
→ Se tiver erro, me avisa!
```

### Mesma carteira em ambos roles
```
→ Hard refresh: Ctrl+Shift+R
→ Limpa cache do navegador
→ Tenta novamente
```
