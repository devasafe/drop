## Comparação: Antes ❌ vs Depois ✅

### ANTES (Sem Socket.io)
```
Cliente escreve → Salva via API → Outro cliente F5 → Vê msg
                ↓
            2-5 segundos
```

### DEPOIS (Com Socket.io)
```
Cliente escreve → Aparece na tela (otimismo) → Salva no banco → 
Socket.io broadcast → Outro cliente vê em tempo real
                ↓
            < 100ms
```
